# Zemax Digital

🚀 Live Demo: [Website öffnen](https://portfolio-rouge-nine-10.vercel.app/)

Zemax Digital ist eine moderne Portfolio- und Studio-Website für ein Digital Web Studio.  
Die Website wurde mit Next.js, TypeScript, CSS Modules und Framer Motion umgesetzt und präsentiert Leistungen, Projekte, Skills, Kontaktmöglichkeiten, Startpreise sowie rechtliche Unterseiten.

## Features

- Moderne Startseite mit Hero, About, Benefits, Services, Process, Trust, Skills, Projects, FAQ und Contact
- Unterseiten für About, Services, Skills, Projects und Contact
- Projekt-Detailseiten im Case-Study-Stil
- Leistungen-Seite mit Startpreisen und Paketübersicht
- Responsive Mobile Dock Navigation
- Animierte UI-Elemente mit Framer Motion
- CSS Modules für komponentenbasierte Styles
- SEO-Metadaten, Sitemap und Robots
- Eigenes Logo, Favicon und Webmanifest
- Eigene 404-Seite
- Impressum- und Datenschutz-Seite
- Scroll-to-Top Button

## Tech Stack

- Next.js
- React
- TypeScript
- CSS Modules
- Framer Motion
- Git / GitHub
- Vercel
